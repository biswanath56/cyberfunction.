# CyberFiction | Biswanath

CyberFiction is a modern scroll-based animated web experience built using GSAP and Locomotive Scroll.  
The project demonstrates smooth scrolling, frame-by-frame canvas animation, and pinned scroll sections to create an immersive storytelling interface.

## 🚀 Features
- Smooth scrolling using Locomotive Scroll
- GSAP ScrollTrigger animations
- Frame-by-frame canvas animation (300 image sequence)
- Pinned sections with scroll effects
- Responsive design (Mobile + Desktop)
- Modern UI with animated typography

## 🛠 Technologies Used
- HTML5
- CSS3
- JavaScript
- GSAP
- ScrollTrigger
- Locomotive Scroll

## 🎯 Purpose
This project was built as a creative portfolio experiment to explore advanced scroll animations and interactive web design techniques.
